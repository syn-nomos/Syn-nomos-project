# Semi-Automated Annotation Framework

A Streamlit-based application for efficient Named Entity Recognition (NER) dataset creation.

## Documentation

- **[User Guide & Installation](INSTRUCTIONS_USER_GUIDE.md)**: Setup, running the app, and adding your custom model.
- **[System Architecture](SYSTEM_ARCHITECTURE.md)**: How the Hybrid Predictor and Active Learning work.

## Quick Start

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   streamlit run app/Data_Loader.py
   ```
