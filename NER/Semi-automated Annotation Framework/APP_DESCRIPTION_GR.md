# Περιγραφή Εφαρμογής & Αρχιτεκτονικής

Η ανάπτυξη αξιόπιστων μοντέλων Μηχανικής Μάθησης στον νομικό τομέα προϋποθέτει την ύπαρξη εκτενών και υψηλής ποιότητας συνόλων δεδομένων, η δημιουργία των οποίων αποτελεί μια διαδικασία εξαιρετικά χρονοβόρα και επιρρεπή σε σφάλματα. Για την αντιμετώπιση αυτής της ανάγκης, αναπτύχθηκε ένα εξειδικευμένο πλαίσιο ημιαυτόματης επισημείωσης (Semi-Automated Annotation Framework) που υιοθετεί την προσέγγιση "Human-in-the-Loop", επιτρέποντας στον χρήστη να επιβλέπει και να διορθώνει τις προβλέψεις του αλγορίθμου σε πραγματικό χρόνο. Ο πυρήνας της εφαρμογής βασίζεται σε έναν καινοτόμο Υβριδικό Μηχανισμό Πρόβλεψης, ο οποίος συνδυάζει τη σημασιολογική κατανόηση των σύγχρονων γλωσσικών μοντέλων (Transformers) με την ακρίβεια των μεθόδων ασαφούς ταύτισης (Fuzzy Matching), επιτυγχάνοντας έτσι την ταχεία και συνεπή παραγωγή δεδομένων εκπαίδευσης (Gold Standard Datasets) απαλλαγμένων από θόρυβο και αντιφάσεις.
